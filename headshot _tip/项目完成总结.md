## 🚀 立即可用

### 方案 A：测试版本（推荐新手）

1. **编译项目**
   ```bash
   双击 build.bat
   ```

2. **修改部署脚本**
   - 编辑 `deploy.bat`
   - 修改第 8 行的游戏路径
   ```batch
   set "GAME_PATH=你的游戏路径"
   ```

3. **部署到游戏**
   ```bash
   双击 deploy.bat
   ```

4. **测试功能**
   - 启动游戏
   - 启用 Mod
   - 游戏中按 F8/F9 测试

### 方案 B：完整开发（高级用户）

1. **反编译游戏代码**
   - 参考 `QUICKSTART.md`
   - 使用 ILSpy 查找爆头检测方法

2. **修改 Harmony Patch**
   - 在 `ModBehaviour.cs` 中
   - 取消注释对应的 Patch
   - 根据实际方法调整

3. **编译测试**
   ```bash
   build.bat
   deploy.bat
   ```
