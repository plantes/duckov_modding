# RunWhileEating - 吃东西时可以跑

**让你在使用消耗品时保持正常跑步速度！**

---

## 📖 功能简介

在原版游戏中，使用食物、药品等消耗品时会强制降低移动速度或无法移动。这个 Mod 移除了这个限制，让你可以：

- ✅ **使用食物时正常跑动** - 边吃边跑，不再蹲在原地
- ✅ **使用药品时正常跑动** - 边治疗边跑，战术撤退更灵活
- ✅ **使用饮料时正常跑动** - 边喝边跑，保持战斗节奏
- ✅ **100% 正常速度** - 不降速，和平时跑步一样快
- ✅ **简单易用** - 安装即生效，无需配置

---

## 🎮 适用场景

- **战斗中急需补给** - 不必停下来吃东西，可以边打边补
- **探索时快速行动** - 路上边走边吃，节省时间
- **PVP 对战** - 更灵活的战术选择，提升生存率
- **赶时间** - 任何需要快速移动的场景

## 💾 安装方法

### 方式 A：自动安装（推荐）

1. 下载 `RunWhileEating_v1.0.zip`
2. 解压到任意位置
3. 编辑 `deploy.bat`，修改游戏路径（第10行）
4. 双击运行 `deploy.bat`

### 方式 B：手动安装

1. 下载 `RunWhileEating_v1.0.zip`
2. 解压，得到 `RunWhileEating` 文件夹
3. 复制整个文件夹到：`{游戏目录}\Duckov_Data\Mods\`
4. 启动游戏
5. 主菜单 → Mods → 启用"吃东西时可以跑"
6. 重启游戏

---

## 🔧 使用方法

### 基本使用

安装并启用 Mod 后，直接游玩即可！使用任何消耗品时自动生效，无需额外操作。

### 调试功能

- **F11** - 显示 Mod 状态信息（查看日志）

### 查看日志

如需查看详细日志：
1. 游戏目录：`%USERPROFILE%\AppData\LocalLow\TeamSoda\Duckov\Player.log`
2. 搜索 `[RunWhileEating]` 查看 Mod 日志

## 🐛 故障排查

### Mod 不显示在游戏中
- 检查文件夹名称是否为 `RunWhileEating`
- 检查是否包含所有必需文件（dll、ini）
- 确认放在正确的 Mods 目录

### 功能不生效
1. 游戏中按 F11 查看 Mod 状态
2. 查看日志文件，搜索错误信息
3. 确认 Mod 已在游戏中启用
4. 重启游戏

### 游戏崩溃或卡顿
- 尝试禁用其他 Mod
- 检查游戏版本是否兼容
- 查看日志文件中的错误信息
